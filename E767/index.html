Project Overview

This project was designed to guide first-year and international college students through the process of finding sources in information. In order to facilitate the learning, the module consists of three objectives, each having its own module:

  1. Identify descriptors used to locate information resources
  2. Use one or more descriptors to find sources in the catalog
  3. Use one or more descriptors to find sources in the discovery tool SUMMON

The content presented in this reusable learning object (RLO), which covers the first objective on the list, aims to teach students about the descriptors used by databases to locate information by introducing them to academic articles and their parts, and how they can be used to find additional sources of information. At the end of the module, learners are presented with a quiz to assess their understanding of the concepts presented.
