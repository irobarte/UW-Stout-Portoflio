Enabling Objective Matrix

UNIT: Understanding Library Services and Library Collections
TERMINAL OBJECTIVE: Users will identify the different library services and different library collections.
PRE-INSTRUCTIONAL STRATEGY: Read introduction in tutorial.
Enabling Objective Level on Bloom's Taxonomy Content Structure Learner Activity Delivery Method
Classify and identify the various library collections held within the library Understanding, Remembering Concept Use library website to identify various collections (e.g. Special Collections, Art Collection) and the focus of each Self-paced activity, Individual activity
Explain how library collections are organized. Understanding Principle & Rule Use a floor plan of the library to identify specialized collections (e.g. Periodicals, Reference) within the library collection Self-paced activity, Individual activity
Describe and identify the various library services. Remembering Concept Use library website to identify the services available at the library Self-paced activity, Individual activity
Demonstrate proper use the LOC Classification System. Applying Principle & Rule Examine call number examples and sort into correct order Self-paced activity, Individual activity, Sorting Activity
Identify search tools located on the library website. Remembering Concept Use library website to locate and identify search tools (e.g. Basic Search, Catalog Search) Self-paced activity, Individual activity
